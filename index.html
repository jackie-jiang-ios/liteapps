桌面清理 — 整理你的桌面文件

为什么这么快？

Alight 采用创新的 JS + Native Bridge 三层架构：

  • UI 层 (Vue 3 + Element Plus)
    轻量 Web 渲染，跨平台一致体验，开发效率高
  • Bridge 层 (IPC 通信)
    高效消息桥梁，毫秒级调用 Native 能力，异步非阻塞
  • Native 层 (Objective-C)
    直接调用 macOS 系统 API，文件扫描、进程管理等极致性能

这种分层架构让 Alight 既拥有 Web 开发的灵活性，又具备原生应用的性能优势。安装包仅 ~10MB，内存占用极低，远优于传统 Electron 方案。
